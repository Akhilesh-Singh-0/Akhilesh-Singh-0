## 🚀 Featured Project

### 🌟 Luminary — Personal Knowledge Management System
> Save what matters. Access it anywhere. Share it instantly.

A full-stack application to store and organize YouTube videos, tweets, articles, and links in one place — with a public sharing system.

| | |
|---|---|
| 🔗 Live | https://luminary-nu.vercel.app |
| 💻 Repo | https://github.com/Akhilesh-Singh-0/luminary |

**Stack:** React · TypeScript · Node.js · Express · MongoDB · JWT · Tailwind · Vercel · Render

- Designed modular backend architecture with clear separation of routes, controllers, and middleware
- Implemented secure JWT authentication with protected API endpoints
- Built public sharing system using unique URL-based access — no authentication required
- Structured REST APIs to efficiently handle multiple content types (videos, tweets, articles)
